This paper provides a detailed analysis of the IDF file architecture generated by mainstream
international petroleum and chemical pipeline 3D design software. Utilizing Python modules such
as Numpy/Matplotlib and the Deap genetic algorithm, the study conducts data cleansing and
analysis on the physical and spatial data of components within the files. Algorithms for material
statistics, weld data, model replication, and work scheduling have been developed and applied in
the data preparation of multiple projects. This work addresses the lack of foundational data
interchange file support in the digitalization of BIM for petroleum and chemical pipeline projects,
oﬀering a reference for the exploration of digital construction in small and medium-sized
petrochemical enterprises.

i.
在焊工排程的初始化阶段，根据焊缝的三维坐标数据建立了详细的工区基础数据。每个
