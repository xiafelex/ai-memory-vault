# 马来西亚生物柴油项目与 Technip 分工策划主档案

一个面向 2026 年 4 月 25 日交流场景的项目协作主档案。

它解决的是这件事：

- 需要把马来西亚生物柴油项目与 Technip 的合作模式整理成一版可直接交付、可直接交流的正式文本
- 需要把 Engineering / Procurement / Construction / Local Approval 的边界说清楚，避免会议中只停留在泛泛表态
- 需要用易高项目的原始采购记录和已验证执行经验，证明这套分工不是设想，而是有落地基础的执行模式
- 需要先形成概念性交付版本，为后续补充主材分交表、品牌清单和责任矩阵留下接口

## 设计原则

- 主档案优先：先形成一份可独立阅读的完整主档案，再补表格和附件
- 事实优先：所有关键判断尽量回到易高项目的台账、原始合同名称和已验证执行经验
- 分层表达：把总口径、采购结构、专业分工和待补资料分开，不混在一起
- 交付导向：内容服务于对外交流和内部统一口径，而不是保留对话痕迹
- 可扩展：本稿先做概念交流版，后续可继续扩展成会议版、英文版和表格版

这份文档是当前阶段的“概念交流版主档案”。

## 总体判断

基于易高项目现有资料，当前可以先形成以下总体判断：

- 易高项目已证明，NCC / 南化建体系实际组织过一整套覆盖工艺、储运、电气、仪表、消防、预制和现场实施的综合采购体系，而不是仅承担若干零散材料采购
- 这套采购体系不是单一路径，而是国内采购、国内预制配套采购和马来西亚属地采购三线并行
- 因此，本次新项目不宜仅按“甲供还是乙供”来讨论分工，更合理的方式是按“系统责任层、国内制造层、国内预制层、属地落地层”四层执行面来划分
- Technip 的核心价值在于总体设计、ISBL、关键工艺系统集成和性能责任
- NCC 的核心价值在于把中国供应链、国内预制、跨境物流、海外施工和本地审批支持真正组织落地
- 新项目最值得复制的，不是某个单独品牌，而是“国内主材采购 + 国内预制 + 属地配套采购 + 海外施工实施”的混合执行模式

## 易高项目给出的执行证据

基于 `NCC_PUR_PurchaseContract (1).xlsx` 中与易高项目相关的采购记录，可以先反推出以下事实：

- 易高相关采购共分为三条记录主线：
- `马来西亚易高项目`，74 份合同，约 7719 万元，代表马来西亚现场及属地采购
- `马来西亚易高国内采购项目`，287 份合同，约 2.59 亿元，代表国内主材和设备采购
- `马来西亚易高项目国内预制项目`，58 份合同，约 7336 万元，代表与国内预制深度绑定的采购
- 三类合计共 419 份合同，总额约 4.10 亿元

这说明易高项目已经验证了以下组织方式：

- 国内主材和关键制造资源前置锁定
- 国内预制与现场安装同步策划
- 马来西亚属地采购补足电缆、消防、弱电、防火、机具和现场响应型物资
- 采购、预制、物流、现场施工并不是串行关系，而是分层联动关系

从结构上看，还可以支持以下判断：

- 国内采购和国内预制合计占比约 81.15%，说明新项目中大量大宗主材和预制相关物资应优先纳入 NCC 执行范围
- 属地采购占比约 18.85%，说明马来西亚本地采购不是零星补项，而是现场适配和本地合规的重要组成部分
- 采购方式以单一来源和询价为主，体现的是项目型执行逻辑，即交期、接口、可追踪、可落地优先

## 易高项目采购范围的概念性汇总

以下总结基于采购台账中的合同名称这一层原始文本，目的是先形成一版概念级主档案描述。

从概念上看，易高项目已覆盖的主要采购类别至少包括：

- 管道主材
- 管件法兰
- 阀门
- 储罐及附件
- 电缆
- 桥架及母线
- 开关柜及配电
- 仪表材料
- 弱电及控制
- 钢结构及防火
- 保温及油漆
- 消防系统
- 焊材及预制耗材

如果把土建建材、施工机具和零星辅材一并计算，则可以认为易高项目已经形成了一套覆盖工艺、储运、电气、仪表、消防、预制和现场实施的综合采购体系。

从专业构成上，当前最有代表性的几类物资为：

- 管道类：不锈钢管、碳钢管、工艺管件、法兰、地管、仪表管阀件等
- 阀门类：700 / 800 / 900 单元工艺高压阀门、中低压阀门、OSBL 区域工艺阀门、消防阀门等
- 储罐类：罐板、型材、储罐附件、储罐计量系统、储罐钢格板、储罐保温及油漆等
- 电气类：高压电缆、中低压电缆、控制电缆、仪表电缆、桥架、低压柜、中压柜、变压器、UPS 等
- 仪表及控制类：温度仪表、防爆接线箱、CEMS、监控与弱电系统等
- 消防及安全类：消防泵、稳压泵、消防水箱、喷淋装置、防火涂料等
- 预制及施工配套类：焊材、板材、油漆、脚手材料、施工机械、行车、电动葫芦等

从采购落点上，可以形成如下概念划分：

- 国内采购为主的物资：不锈钢管、碳钢管、管件、阀门、储罐附件、桥架、低压柜、中压柜、变压器、焊材、油漆等
- 国内预制配套采购：第二批乙供不锈钢管、碳钢管、管件法兰、储罐型材、焊材、板材、管道油漆、储罐油漆等
- 国外属地采购：高低压电缆、仪表电缆、控制电缆、消防泵、监控弱电、防火涂料、部分钢结构安装配套、施工机械、电梯等

从品牌和厂家分布上，可以先形成概念性结论：

- 管道主材代表厂家主要集中在国内，如浙江久立、江苏武进、中兴能源、杭州海泰、扬州管件、合肥实华、江苏顺通等
- 阀门类以国内厂家为主，如江苏大阀、凯斯通、安徽铜都等
- 储罐附件和计量系统主要落在国内厂家，如湖北天信、江苏泽拓等
- 电气大件以国内供应链为主，如镇江西门子母线、金海新源、南京凯拓、安徽天康等
- 电缆和部分现场电气材料更多体现属地采购特征，如 XIN DAO、MAGNUM CABLE、PTH SAFETY 等
- 消防、防火和弱电监控也有明显属地采购特征，如 CRS、RUSHOE、AI SECURITY、ONE GASMASTER 等
- 油漆、防腐、预制耗材则表现为国内品牌和现场使用并行，例如佐敦涂料、江苏瑞瓦、南京昌泰、南京柯威等

因此可以形成一个明确判断：

- 易高项目中由 NCC / 南化建体系实际组织采购的，不是若干零散主材，而是一整套按专业分层、按执行场景分布、按国内与属地协同配置的综合采购体系
- 国内侧更像“主材与预制中心”，国外侧更像“现场落地与本地适配中心”

## 项目协作主框架

建议本次与 Technip 交流时采用以下主框架：

- Technip 负责总设计、ISBL 设计、关键工艺系统集成、关键设备技术接口和系统责任
- 华东院负责 OSBL 设计、公用工程和配套系统设计
- NCC 负责乙供采购、甲定乙采执行、土建及地管相关采购、中国境内催交与监造、检验、发运、到马来西亚的物流组织、现场施工及本地审批配合
- 对性能责任敏感、系统集成性强、长周期风险高的关键设备，由 Technip 主导采购更稳妥

## 分工建议

### 1. Engineering

- Technip：总体方案、ISBL、关键工艺单元、关键设备 datasheet、性能责任
- 华东院：OSBL、公用工程、储运配套、厂外接口、总图配合
- NCC：施工可实施性反馈、预制边界建议、运输限界反馈、吊装和现场安装条件反馈

建议在交流中按单元和系统拆开说明：

- ISBL：按 700 / 800 / 900 等工艺单元拆分
- OSBL：按 hydrogen、plant air、water treatment、natural gas、steam、fire water、tank farm、loading / unloading、jetty pipeline 等拆分

### 2. Procurement

建议采用“两层表达、四层执行”的方法。

对外交流时，口径可简化为：

- A 类：Technip 主导采购
- B 类：甲定乙采
- C 类：NCC 主导采购

内部策划和后续分交表则建议按以下四层执行面展开：

- 系统责任层：关键工艺包、关键长周期设备、对性能考核影响大的设备，由 Technip 主导
- 国内制造层：不锈钢管、碳钢管、管件、阀门、储罐附件、桥架、柜类等中国供应链成熟物资，由 NCC 组织国内采购
- 国内预制层：与预制深度绑定的管材、管件、焊材、油漆、板材等，由 NCC 结合预制厂统筹
- 属地落地层：电缆、防火涂料、消防泵、弱电、装修、临建、施工机械等贴近本地规范和现场响应的物资，由马来西亚团队采购

更适合 Technip 主导的物资：

- 对核心工艺性能有直接影响的关键成套设备
- 长周期且系统责任敏感的设备
- licensor、融资、保险或业主条件要求更强的设备
- reformer、boiler、部分关键 exchanger、关键 rotating package

更适合 NCC 主导的物资：

- 大宗材料
- 管道主材、管件、支吊架、阀门、紧固件
- 土建、地管、给排水、消防常规材料
- 钢结构附材、防腐保温、电仪散材
- 业主未限定欧洲品牌、且中国供应链成熟的常规动静设备

适合甲定乙采的物资：

- 技术标准要统一，但采购执行更需要灵活性和催交能力的物资
- 品牌可 shortlist，但实际制造、催交、检验、发运更适合由 NCC 执行的物资

建议固化成一句总原则：

- Technip 决定系统责任，NCC 组织执行落地，采购边界按照性能责任、制造成熟度、预制耦合度和属地适配性来划分

### 3. Construction

NCC 在交流中应重点突出以下能力：

- 中国预制 + 海外现场安装的执行模式
- 多专业施工组织能力，包括土建、钢结构、设备、工艺管道、电仪
- 大件运输和大型吊装能力
- 高标准焊接质量控制能力
- 中方与属地团队协同能力

建议作为会议亮点直接表达：

- 80 名焊工可实现日均约 1600 dia-inch 焊接产出
- RT rejection rate 可控制在 1% 以下
- 多材质、ASME 体系、第三方见证、全过程追溯均已有项目经验

### 4. Local Approval

这部分建议明确归入 NCC 的差异化价值，而不是只作为施工附属工作。

建议重点强调：

- NCC 可协助推进 Bomba certificate
- NCC 可协助推进 CCC
- NCC 可协助推进 JKKP / DOSH 相关审批
- NCC 可协助推进 boiler、natural gas、jetty import / export pipeline 相关许可
- NCC 可协助推进 SAJ、TNB 等现场投运相关证书和接口工作

## 围绕四个问题的当前判断

### 1. 主材分交表和最终品牌

当前可以确认的是：

- 现有资料已经能够支撑采购组织模式判断
- 现有资料仍不能直接支撑最终品牌定版
- 当前尚未看到易高项目主材分交表原件，但采购台账已经足以反推出大类边界和专业分布

当前建议做法：

- 先按设备类别和责任逻辑与 Technip 交流
- 先按“系统责任层 / 国内制造层 / 国内预制层 / 属地落地层”交流分交逻辑
- 在没有原始分交表和最终品牌清单前，不承诺具体品牌和最终边界

### 2. 运输模式、问题和建议

已验证的模式为：

- 国内段：工厂发运、订舱、报关、港口操作
- 国际段：海运 / 空运
- 马来西亚段：换单、港杂、清关、陆运到现场

当前建议补充以下原则：

- 运输不是采购完成后的后置动作，而是采购分层时就应确定的前置约束
- 进入“国内预制层”的物资，应在下单前同步确认预制厂加工能力、分段尺寸、包装方式和出海条件
- 进入“属地落地层”的物资，应优先考虑本地规范、供货响应速度和现场安装窗口，而不是单纯比较价格

### 3. 是否需要甲改乙或乙改甲

建议乙改甲的范围：

- 核心工艺包关键设备
- 长周期且责任敏感的关键设备
- 需由 Technip 统一承担性能责任的关键系统

建议甲改乙的范围：

- 大宗材料
- 常规设备
- 施工强接口物资
- 需要高频催交、资料追踪和交付配套的物资

判断原则：

- 系统责任优先
- 施工接口优先
- 长周期优先
- 可执行性优先
- 预制耦合优先

### 4. 品牌问题和替代建议

当前可以明确的是：

- 目前尚未看到易高项目正式的问题品牌清单
- 但从已掌握资料看，风险往往不只来自品牌本身，更来自资料完整性、交期、检验配合和本地合规能力

建议的品牌逻辑：

- 客户未指定欧洲品牌时，中国制造可作为优先选项
- 关键设备采用“双清单”模式：国际品牌备选 + 中国优质品牌备选
- 常规设备更关注海外业绩、交付周期、第三方检验配合度和本地认证适配性

## 建议的会议表达顺序

建议按以下顺序组织 2026 年 4 月 25 日交流内容：

1. 先讲总体判断：本次讨论的不是单点采购，而是 joint execution model
2. 再讲易高经验：易高项目已形成综合采购体系，而不是零散采购
3. 再讲证据支撑：419 份合同、约 4.10 亿元、三线并行采购组织
4. 再讲主要物资构成：管道、阀门、储罐、电气、仪表、消防、预制耗材等主要类别
5. 再讲国内与国外分布：哪些适合国内主材采购，哪些适合国内预制配套，哪些适合属地采购
6. 再讲分工：Engineering / Procurement / Construction / Local Approval
7. 再讲采购分层：系统责任层、国内制造层、国内预制层、属地落地层
8. 再讲四个具体问题：分交、运输、甲乙供调整、品牌替代
9. 最后讲 next step：一周内共同比对分交表和品牌清单第一版
