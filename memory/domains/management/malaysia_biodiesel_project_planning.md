# 马来西亚生物柴油项目分工策划

## 主题定义

这个主档案用于沉淀马来西亚生物柴油项目中，与联合执行模式、采购边界、施工落地、物流清关和本地审批有关的长期可复用判断。

它不是一次性的会议纪要，而是后续继续补充主材分交表、品牌清单、责任矩阵和英文交流材料时的母版。

## 当前稳定框架

### 1. 总体协作模式

建议采用以下总体模式：

- Technip：总设计、ISBL 设计、关键工艺系统集成、关键设备技术接口及系统责任
- 华东院：OSBL 设计、公用工程及配套系统设计
- NCC：乙供采购、甲定乙采执行、土建及地管范围采购、中国境内催交监造、检验、物流组织、现场施工及本地审批配合

### 2. 采购边界判断原则

更适合 Technip 主导采购的物资：

- 对工艺性能、系统联保、成套责任影响大的关键设备
- 长周期且责任边界敏感的关键设备
- 需由工程方统一承担系统责任的关键系统

更适合 NCC 主导采购的物资：

- 大宗材料
- 施工强接口物资
- 国内供应链成熟、催交和交付灵活性要求高的物资
- 业主未指定欧洲品牌的常规设备与材料

适合甲定乙采的物资：

- 技术标准和品牌边界需要统一
- 但制造、催交、检验、发运更依赖国内采购执行能力

### 3. 运输与执行模式判断

当前稳定判断：

- 适合延续“中国采购与预制 + 海运 + 海外安装”的执行模式
- 物流不只是海运问题，更是设计、分段、准证、清关、陆运、吊装一体化问题
- 大件设备运输应前置完成 route survey、道路加固、清障和吊装窗口统筹
- 马来西亚项目中，CIDB、DOSH、CCC、Bomba 等事项应视为项目执行关键路径的一部分

### 4. 品牌与供应商策略

- 客户未指定欧洲品牌时，中国制造可作为优先选项
- 关键设备应采用“国际品牌备选 + 中国优质品牌备选”的双清单模式
- 常规设备更应关注海外业绩、交期稳定性、第三方检验配合度和本地认证适配性，而不是只看品牌知名度
